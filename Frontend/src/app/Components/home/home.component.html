Graph Modeling Application

Easily create, visualize, and manage graph-based data models with our intuitive interface

Graph Manager

Create and manage multiple graph models for your various projects and use cases.

Switch between different graphs to organize your node and connection data.

Basic View

View your graph data in a table-based format for quick analysis and editing.

Create and manage nodes and connections in a structured layout.

Interactive Visualization

Interactive graph visualization with drag-and-drop functionality.

Explore complex relationships visually and interact with your graph model.

Node Management

Create, view, and manage nodes (vertices) in your graph model.

Nodes represent entities in your data model, each with customizable properties and types.

Connection Management

Define relationships between nodes with typed connections (edges).

Connections can have their own properties to describe relationships in detail.

Cosmos DB Integration

Powered by Azure Cosmos DB for scalable graph storage.

All your graph data is securely stored and easily retrievable.

Getting Started

1Create a Graph

Start by creating a graph to organize your nodes and connections.

  1. Navigate to Graph Manager
  2. Click on "Create New Graph"
  3. Enter a unique name for your graph
  4. Click "Create Graph" to save

2Add Nodes to Your Graph

Once you have a graph, you can start adding nodes (entities) to it.

  1. Navigate to Basic View or Interactive View
  2. Select a graph from the dropdown at the top
  3. Open the Tools Panel on the right (if collapsed)
  4. Select the "Nodes" tab
  5. Enter a name for your node
  6. Select a node type from the dropdown
  7. Click "Create Node" to save

3Create Connections Between Nodes

Connect your nodes to create relationships in your graph.

  1. Stay in either Basic View or Interactive View
  2. Open the Tools Panel if collapsed
  3. Select the "Connections" tab
  4. Select a source node from the dropdown
  5. Select a target node from the dropdown
  6. Choose a connection type
  7. Click "Create Connection" to save

4Visualize Your Graph

View and interact with your complete graph model.

  1. Navigate to the Interactive View for visual exploration
  2. Drag nodes to arrange your graph
  3. Click on nodes or edges to view their details
  4. Use the layout controls to arrange your graph automatically
  5. Switch to Basic View for tabular data representation
